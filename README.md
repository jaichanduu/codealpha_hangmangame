🎮 Hangman Game

A simple and interactive Hangman Game developed using Python. This project is created as part of the CodeAlpha Python Programming Internship.

📌 Project Overview

Hangman is a classic word-guessing game where the player tries to guess the hidden word one letter at a time before all chances are lost.

This project demonstrates:

Python programming fundamentals

Loops and conditional statements

String handling

Random word selection

User input handling





✨ Features

1.Random word generation

2.Letter-by-letter guessing

3.Limited attempts system

4.Displays correct and incorrect guesses

5.Simple terminal-based gameplay

6.Beginner-friendly Python project



🛠️ Technologies Used

1.Python 3

2.VS Code

 🎮Game Rules:

1.The player has limited attempts to guess the word.

2.Each wrong guess reduces one chance.

3.The game ends when:

The player guesses the word correctly 🎉

All attempts are used ❌
